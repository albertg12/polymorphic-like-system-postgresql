# Sistema de likes polimórfico (PostrgeSQL)

## Descripción
Este proyecto implementa un sistema de likes donde los usuarios pueden dar likes a posts o comments usando un diseño polimófico.

## objetivo
practicar:
- asociones polimórficas
restricciones directamente en la base de datos.
- uso de COALESCE
- Diseño relacional

## Base de datos
postgreSQL

## Herramientas
- pgAdmin 4
- dbdiagram.io
